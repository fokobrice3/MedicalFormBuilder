Medical Form Builder

Cette application est réalisé pour l'unité d'enseignement INF4188 : Dévoloppement web sémantique.
Le Tp consiste à développer une plateforme de Web sémantique pour la modélisation et la génération de logiciels de santé. L'objectif principal de cet exercice est de renforcer les capacités des étudiants dans le développement d'applications Web sémantiques. A cette fin, nous avons développé une ontologie depuis Protégé et nous l'avons chargé dans GraphDB. Cette ontologie a été ainsi cette présente plateforme de génération automatique de formulaire web spécifique aux logiciels de santé.
